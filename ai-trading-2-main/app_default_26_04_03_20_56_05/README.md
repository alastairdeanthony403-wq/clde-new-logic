# AI Stock Analysis Dashboard

A professional-grade, interactive dashboard that combines real-time stock market data with custom AI-generated trading signals. The application fetches live history using `yfinance` and integrates with a mock AI analysis payload that represents your proprietary engine's output.

## Features
- **Real-time Data**: Fetches 6 months of price history using FastAPI and yfinance.
- **AI Analysis Integration**: Displays custom summary, bullish/bearish signals, and reasoning from your AI system.
- **Dynamic Charting**: Responsive price history charts using Chart.js.
- **Price Target Visuals**: A target range display visualizing the 30-180 day horizon.
- **Modern UI**: Clean, dashboard-style interface built with Tailwind CSS and Lucide icons.
- **Ticker Search**: Easily switch between any valid stock ticker (e.g., AAPL, TSLA, BTC-USD).

## Tech Stack
-   **Frontend**: React (Vite), Tailwind CSS, Chart.js, Lucide Icons.
-   **Backend**: Python (FastAPI), yfinance, Uvicorn.

## Prerequisites
-   **Python 3.8+**
-   **Node.js 18+**
-   **npm** (or yarn)

## Installation & Setup

### 1. Backend Setup
Navigate to the backend directory and install dependencies:
    cd backend
    pip install -r requirements.txt

Run the backend server:
    python main.py

The REST API will be running at `http://localhost:8000`.

### 2. Frontend Setup
In a new terminal, navigate to the frontend directory:
    cd frontend
    npm install

Run the development server:
    npm run dev

The web interface will be available at `http://localhost:3000`.

## How to Use
1.  **Search**: Use the search bar in the top navigation to enter a stock ticker (e.g., `MSFT`).
2.  **Analyze Trends**: Observe the 6-month price chart for historical context.
3.  **Review Signals**: Scan the green (bullish) and red (bearish) signals generated by the AI reasoning engine.
4.  **Target Monitoring**: Check the AI Price Target card to see the projected base price and range.
5.  **Refresh**: Use the refresh icon next to the search bar to fetch fresh market data and re-run analysis.

## Integration Note
To connect your own real AI model:
- Open `backend/main.py`.
- Modify the `get_ai_analysis` endpoint to call your local model script or an external inference API.
- Ensure the JSON response matches the established structure for the frontend to render correctly.

## Troubleshooting
- **CORS Errors**: The backend is configured with `allow_origins=["*"]`. If you deploy to production, update this to your specific domain.
- **yfinance Delays**: If the stock data does not load, yfinance might be experiencing rate limits. Try refreshing after a few seconds or checking your internet connection.
- **Node Modules**: If icons or charts fail to render, run `npm install` again to ensure all peer dependencies are resolved.
